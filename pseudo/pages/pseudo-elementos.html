Pseudo-elementos

Usado para adicionar elementos extras na página, sem a necessidade de declará-lo no HTML, ou para selecionar uma parte de algum elemento.

Identificado por ::palavra-chave

Pseudo-elemento - ::first-letter

Selecionar a primeira letra da primeira linha

Propriedades disponíveis:

  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedades de texto

Pseudo-elemento - ::first-line

Seleciona a primeira linha de um texto

Propriedades disponíveis:

  • color
  • background
  • font
  • Propriedades de Texto

Pseudo-elemento - ::backdrop

Estiliza o plano de fundo de elementos em modo de tela cheia, como dialog ou video
